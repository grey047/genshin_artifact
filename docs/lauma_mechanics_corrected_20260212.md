# 月绽放机制重新理解 (修正版)

**日期**: 2026-02-12  
**修正**: 基于用户反馈重新理解机制

---

## 1. 核心概念澄清

### 错误理解 ❌
- 月绽放有独立的反应倍率 (如3.0)
- 直伤月绽放是Lauma专有
- EnhanceLunarBloom 是普通月绽放加成

### 正确理解 ✅

**月绽放 (Lunar-Bloom) 本质**:
- **不是独立反应**，而是**绽放的变种/强化**
- 由**月兆角色**在场时，草+水触发的特殊绽放
- **主要作用**: 产生"草露"(Verdant Dew/Spirit Envoy)，为角色技能提供资源

**伤害来源** (两种):

| 类型 | 机制 | 触发者 | 倍率来源 |
|------|------|--------|----------|
| **种子伤害** | 月绽放产生的种子爆炸 | 任何角色 | 同超绽放/烈绽放 (3.0) |
| **直伤月绽放** | 角色技能直接造成月绽放伤害 | 特定角色 (Lauma, Nefer等) | 技能倍率 × 精通 |

## 2. 直伤月绽放 (Direct Lunar-Bloom DMG)

### 定义
角色技能**直接造成**的、被标记为"月绽放伤害"的伤害。

### 特点
- 基于**精通**而非等级系数
- 可享受月绽放相关加成
- **不是反应**，是**技能伤害类型**

### 已知能打出直伤月绽放的角色

| 角色 | 技能 | 倍率来源 |
|------|------|----------|
| **Lauma** | E技能长按第二段 | 技能倍率 × 草露数量 |
| **Lauma** | C6召唤物攻击 | 精通 × 185% |
| **Nefer** | (待确认) | (待确认) |

## 3. EnhanceLunarBloom vs 其他加成

### 当前理解

| 属性 | 作用对象 | 类比 |
|------|----------|------|
| **EnhanceLunarBloom** | 月绽放反应伤害 (种子) | 类似妮露的"丰穰之核"加成 |
| **直伤月绽放加成** | 角色技能的月绽放伤害 | 类似技能伤害加成 |

### 具体区分

**种子伤害 (反应)**:
```
伤害 = 等级系数 × 3.0 × (1 + 精通加成 + EnhanceLunarBloom) × 暴击 × 抗性
```

**直伤月绽放 (技能)**:
```
伤害 = 精通 × 技能倍率 × (1 + 直伤加成 + 其他乘区)
```

## 4. Lauma 机制详解

### A3 天赋 (Moonsign Benediction)
- 队伍触发绽放时转为月绽放
- **月绽放基础伤害加成**: 精通 × 0.0175%, 上限14%
- **注意**: 这是给**种子**的加成，不是给直伤

### A1 天赋 (Light for the Frosty Night)
- **月兆·初辉**: 种子可暴击 (15%暴击率/100%暴伤)
- **月兆·满辉**: 种子暴击率+10%，暴伤+20%
- **注意**: 这也是给**种子**的

### C6 命座 (直伤)
- 召唤物造成 **精通 × 185%** 的伤害
- 视为**月绽放伤害** (直伤类型)
- **月兆·满辉**: 直伤 **elevated** (擢升) +25%

## 7. 修复计划调整

### 需要修正的地方

**原理解**:
```rust
// 错误: 把月绽放当作独立反应
let lunar_bloom_damage = level_multiplier * 3.0 * ...
```

**修正后**:
```rust
// 种子伤害 (超绽放类型)
let seed_damage = level_multiplier * 3.0 * (1 + em_bonus + enhance_lunar_bloom) * ...

// 直伤月绽放 (Lauma技能)
let direct_lunar_damage = em * skill_ratio * (1 + elevated_bonus) * ...
```

### 属性命名建议

| 属性 | 用途 |
|------|------|
| `EnhanceLunarBloom` | 月绽放种子伤害加成 |
| `LunarBloomCritRate` | 月绽放种子暴击率 (A1) |
| `LunarBloomCritDMG` | 月绽放种子暴击伤害 (A1) |
| `ElevateLunarBloom` | 直伤月绽放擢升加成 (C6) |
